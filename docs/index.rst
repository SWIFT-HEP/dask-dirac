Introduction
------------

The `dask-dirac`_ package is a Python package for submitting `Dask`_ workflows to the grid via `DIRAC`_ (the interware).
The implementation relies on DIRAC's HTML REST API and thus requires only minimal dependencies (e.g. `requests`_).
